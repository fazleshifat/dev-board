3.getElementByName()

6.querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between three of them is
1.innerHTML gives the whole structure of and html content with all tags and text.
2.innerText gives the visible text only from the specific html tags and element which has been defined.
3.textContent gives the whole html structure with all tags,elements, all visible and invisible text.
There also three different use of this three method.

Question-3: What is event delegation in the DOM?

In JavaScript there is a method call event delegation which is used to handle event.We use it on a parent div to handle the activities of an child element by target method.
It's basically used in some way instead of event listener.

Question-4: What is event bubbling in the DOM?

Event bubbling is a method in the JavaScript DOM where if we click on a child div, first it will select the targeted element that means the child element,
after that it will select the parent div,after that it will select the second parent div,then it will select the third parent div.LIKE A LOOP
it will select from the last div(selected element) to the main div.It's call bubbling.
